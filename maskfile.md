# Grindshell Frontend

## solidui

> Interact with the solidui cli

```powershell
yarn solidui-cli --help
```

### init

> Init solidui config files

```powershell
yarn solidui-cli init
```

### add

> Interactively add a component

```powershell
yarn solidui-cli add
```

## run

### desktop

```bash
yarn tauri dev
```

### web

```bash
yarn vite
```
